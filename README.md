# BoGSL
A grid-based Board Game Specific language 
created by group 5 for a project from Software Language Engineering at the Vrije Universiteit Amsterdam in 2026

## Project layout
- `pom.xml`
- `META-INF/RASCAL.MF`
- `src/main/rascal/*.rsc`
- `src/main/rascal/*.dsl`

## Build
Run from repository root:

```sh
mvn package
```
